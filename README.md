# Aurora Café — Landing Page Premium

## Descripción
**Aurora Café** es una landing page premium para una cafetería de especialidad.

El proyecto fue diseñado con una estética cinematográfica y moderna, enfocada en:
- Experiencia visual elegante
- Animaciones suaves
- Diseño responsive
- Interacciones dinámicas
- Performance optimizada
- UI/UX premium

Incluye secciones como:
- Hero principal
- Historia del café
- Especialidades
- Granos del mes
- Repostería
- Información de contacto y visita

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (vanilla)
- Intersection Observer API
- GSAP + ScrollTrigger
- Lenis (smooth scroll)

## Características principales

### Hero cinematográfico
- Animaciones premium de entrada
- Parallax suave
- Efecto tipo "cinema reveal"
- Estadísticas dinámicas

### Diseño UI moderno
- Estética minimalista y elegante
- Tipografías premium
- Layout responsive
- Colores cálidos inspirados en café de especialidad

### Animaciones avanzadas
- Reveal animations al hacer scroll
- Curtain reveal effects
- Hover effects suaves
- Scroll animations optimizadas

### Experiencia interactiva
- Navbar dinámica
- Highlight automático de secciones
- Estado "Abierto / Cerrado" en tiempo real
- Horarios con barras animadas

## Cómo ejecutar / ver el sitio
Opción 1 (rápida):
- Abrí `index.html` en tu navegador.

Opción 2 (recomendada): servidor local.

Si tenés Python instalado:
```bash
python -m http.server 5500
```
Luego abrí:
- http://localhost:5500/index.html

## Funcionalidades JavaScript
- Navbar inteligente (scroll shrink + sección activa)
- Sistema de animaciones por scroll (reveal)
- Parallax en el hero
- Cálculo dinámico del estado del local (abierto/cerrado)
- Animación progresiva de barras de horario
